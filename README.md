# Autonomous Mobile Robot – PIC16F877A

Autonomous obstacle-avoidance robot built from scratch as part of a university 
project at the Faculty of Electrical Engineering, University of Sarajevo.

## Overview

The robot navigates autonomously using three Sharp IR distance sensors and a 
priority-based obstacle avoidance algorithm. Motor control is implemented via 
custom H-bridge drivers and PWM signals generated by the PIC16F877A 
microcontroller. The entire system, including H-bridge circuits and power 
supply, is integrated on a custom-designed PCB.

## Hardware

- PIC16F877A microcontroller
- 3x Sharp GP2D12 IR distance sensors (front, left, right)
- Custom H-bridge motor drivers (MOSFET-based: FQP13N06L + FQP17P06)
- 2x DC motors (RE 385)
- Custom PCB (designed in Proteus)
- Power supply: 9V (logic) + 12V (motors)

## Features

- IR-based obstacle detection with ADC conversion
- Priority-based navigation algorithm (alternates left/right priority)
- PWM motor speed control (90% duty cycle)
- Timer0 interrupt-based timing for rotation control
